Sistema de Cadastro de Utilizadores em JavaFX

📌 Sobre o Projeto

Este projeto foi desenvolvido no âmbito da disciplina de Sistemas Distribuídos e Paralelos I.

Foi-nos solicitado a criação de um Formulário de cadastro , utilizando a tecnologia JavaFX, com o ambiente de desenvolvimento NetBeans, e a versão Java 8.

O sistema permite realizar o cadastro de utilizadores através de uma interface gráfica amigável, armazenando os dados inseridos em ficheiro binário para posterior consulta e persistência das informações.

🚀 Tecnologias Utilizadas

Java 8
JavaFX
NetBeans IDE
Programação Orientada a Objetos
Manipulação de Ficheiros Binários

🎯 Funcionalidades

Cadastro de utilizadores
Interface gráfica com JavaFX
Validação de campos do formulário
Armazenamento de dados em ficheiro binário
Leitura dos dados gravados
Organização do código em classes

🖥️ Interface do Sistema

O sistema apresenta formulários contendo campos para inserção de dados pessoais do utilizador, como por exemplo:

Nome
Data de Nascimento
Género
Número de Identificação
Outros campos solicitados no trabalho
💾 Armazenamento

Os dados cadastrados são guardados em ficheiros binários, permitindo:

Persistência dos dados após fechar o programa
Leitura posterior dos registos
Maior organização no armazenamento
